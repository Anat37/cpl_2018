Код программы состоит из описания функций не разделенных ничем.
Исполнение будет начинаться с функции main.
Функция должна определяться раньше чем ее вызов.
Выполнение завершается по команде exit

Определение функции:
ключевое слово def (rec) [имя] [количество аргументов] (args)
rec надо указывать для рекурсивных функций
Определение функции заканчивается ключевым словом enddef

Все переменные имеют тип int или являться адресом строки
Чтобы объявить глобальную переменную типа int: gl_var [name] [value]
Чтобы объявить строку: var_string [name] [value]
Все строки являются глобальными
Чтобы объявить локальную переменную int: var [name] [value]

Значения регистров, доступных для использования: reg1, reg2, reg3
Ввод-вывод:
Вывести значение лежащее в регистре: print [regX]
Считать значение в регистр: scan [regX]
Вывести строку: prints [name]

Команды:
mov [arg1] [arg2] - скопировать значение из arg1 в arg2
arg1 - переменная, регистр, значение
arg2 - переменная, регистр

sum -сложить значения в регистрах 1 и 2, результат будет в регистре 3
dec -вычесть аналогично
more -reg1 > reg2 аналогично
equal, or, and - аналогично

call [func name] (args) - вызов функции
аргументами могут быть имена переменных или значения int
для выхода из функции можно использовать return

if 
[команды]
else
[команды]
endif
секцию else можно пропустить
if использует значение первого регистра
